# Textual

Entrypoint to various notes on Textual.

\>>> **[tutorial/tutorial.md](tutorial/tutorial.md)**<br/>
Notes based on the official tutorial at [[https://textual.textualize.io/tutorial/](https://textual.textualize.io/tutorial/). Topics: installation, running, a basic application (with Buttons, TextArea and DataTable), bindings and styling.

\>>> **[guide/app-basics/readme.md](guide/app-basics/readme.md)**<br/>
Application basics from [https://textual.textualize.io/guide/app/](https://textual.textualize.io/guide/app/). Notes on events, event handlers, widgets, mounting, exiting and suspending.

\>>> **[widgets/tree.md](widgets/tree.md)**<br/>
How to work with the Tree widget. Includes initialization, two ways to compose a tree (one using context managers), events like highlighting and selecting, adding a node, deleting an node and resetting the entire tree.
